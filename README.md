## Compatibility

This mod has only been tested on Satisfactory Version 1.2. It might work on lower versions, but I'm not confident it will.
Unfortunately I don't have the hardware to test this mod on Linux, but it might work.
Unfortunately Trinkets in Satisfactory are client-sided, meaning other players can't see them in multiplayer, but this also means you can join any server with this mod installed without any issues.
